Show Discord thread notifications when a queued message is edited or removed.

When a user edits a message that is in the local queue, the thread now shows
`⬦ **username** edited queued message` or `⬦ **username** removed message from queue`.

The queue confirmation message also tells users they can edit the message to update it:
`Queued at position N. Edit your message to update it in queue`.
